More about Fashion Recommendation System

This project represents a Fashion Recommender System implemented using a pre-trained ResNet50 model for image feature extraction and k-nearest neighbors for similarity recommendations. The recommender system allows users to upload an image, and based on the extracted features, it identifies and displays the closest fashion items. The system is built using TensorFlow and utilizes Streamlit for creating a web interface.

The code is divided into two parts:

Training Script:

  • Imports necessary libraries and sets up a pre-trained ResNet50 model for feature extraction.
  • Defines a function to extract features from images and saves them in the "embeddings.pkl" file.
  • Creates a list of file paths and saves it in the "filenames.pkl" file.
  • The script is used to preprocess the training data and store the features and file paths for later use.

Streamlit Application:

  • Imports required libraries and sets up the Streamlit web application.
  • Loads the pre-trained ResNet50 model and extracts features from the uploaded image.
  • Uses k-nearest neighbors to recommend the closest 5 fashion items based on the extracted features.
  • Displays the uploaded image and the recommended fashion items in a visually appealing layout.
  • Allows users to interactively explore and discover fashion items similar to their preferences.

The Streamlit application provides a user-friendly interface for the Fashion Recommender System, enhancing the online shopping experience by offering personalized suggestions. Users can upload an image and receive tailored recommendations for fashion items. The code also includes some styling elements using CSS to improve the visual appeal of the web interface.
